Gmail integration

Connect receipts using Google OAuth

1 Add credentials.json

Create OAuth credentials in Google Cloud and place the downloaded file in this project folder.

2 Authorize Gmail

Click Sync Gmail. A browser window opens once so you can approve read-only Gmail access.

3 Show reminders

The app scans payment emails, stores detected subscriptions, and displays renewal reminders.

Google Cloud setup: Enable Gmail API, create an OAuth Client ID for a Web application, add this app's /oauth2callback URL as an authorized redirect URI, download the JSON file, rename it to credentials.json, and keep it beside app.py.
To allow Gmail accounts other than test users: In Google Auth Platform, set Audience to External and publish the app for production. Because this app requests read-only Gmail data, Google may require OAuth app verification before it can be used publicly. While the app remains in Testing, only accounts listed under Test users can authorize it.
