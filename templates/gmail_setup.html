Gmail integration

Connect receipts using Google OAuth

1 Add credentials.json

Create OAuth credentials in Google Cloud and place the downloaded file in this project folder.

2 Authorize Gmail

Click Sync Gmail. A browser window opens once so you can approve read-only Gmail access.

3 Show reminders

The app scans payment emails, stores detected subscriptions, and displays renewal reminders.

Google Cloud setup: Enable Gmail API, create an OAuth Client ID for a Desktop app, download the JSON file, rename it to credentials.json, and keep it beside app.py.
If Google shows Error 403 access_denied: Open Google Cloud Console, select this project, go to OAuth consent screen or Google Auth Platform, open Audience, then add your Gmail address under Test users. Use the same Gmail account when approving access, then click Sync Gmail again.
